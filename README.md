# 개요
- java swing, JFrame, Jpanel로 몬스터 잡기
  
----

# 기능 구현 내용
1. 직업 선택 후 몬스터 잡기 게임 시작
2. Q, W, E, R, space bar로 스킬 구현
3. 스킬 사용시 및 공격시 이미지 움직임
4. 턴마다 공격하고 쓰러뜨리면 다음 스테이지
5. HP, MP 바 구현 및
6. 스테이지 올라갈수록 몬스터 체력 및 공격력 높아짐
